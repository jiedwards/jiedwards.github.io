Automated Meal Planner

This project is a work-in-progress, it's end goal is to simplify meal planning. At the current moment, a user can register and login to their account, this has been developed with the use of Passport and BCrypt in Node.js. The current user privileges allow a user to add custom foods into the db. An inner JavaScript application has been built to determine the correct amount of calories for a user based on some details from the user (Mifflin St Jeor Equation is implemented).
The next steps for the project are:
  • Implement Python to sort through the data and execute an import of the data into the MongoDB food database.
  • Implement algorithms, logic and GET requests to retrieve the food information in a sensible way.
    • This is a private project, contact me if you wish to see the code.
