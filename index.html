Bytefit - Automated Meal Planner

The project is hosted bytefit.herokuapp.com
This application allows a user to generate and store calorie intake recommendations, which are then supplemented by one-click generated meal plans. An inner JavaScript application has been built to determine the correct amount of calories for a user based on some required details (Mifflin St Jeor Equation is implemented). User accounts are implemented with the use of Spring Security and BCrypt in Java.
The current privileges of a user with an account are:
  • Save Meal Plans to PDF
  • Dietary Choices e.g. Vegan, Vegetarian, Ketogenic, Paleo etc
  • Store caloric calculations

    • The major next step for the project is to:
  • Replace the requirement of the meal-planning API by developing a database with all the required food/recipe information and relationships.
